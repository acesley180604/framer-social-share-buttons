# Social Share Buttons - Framer Plugin

Add configurable social media sharing buttons to your Framer site.

## Supported Platforms

Facebook, X (Twitter), LinkedIn, Pinterest, Reddit, WhatsApp, Telegram, Email, Copy Link

## Features

- 3 button styles: Icon only, Icon + Text, Text only
- 4 button shapes: Square, Rounded, Circle, Pill
- 4 layout modes: Horizontal, Vertical, Floating sidebar, Inline
- Hover animations: Scale, Lift, Glow, Rotate
- Custom colors (per-platform brand colors or unified theme)
- Share text/URL override with OG preview
- Share analytics tracking
- Self-contained embed code generation

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Usage

1. Open the plugin in Framer
2. Configure platforms, styles, layout, and animations
3. Preview your buttons in the Preview tab
4. Copy the embed code from the Embed tab
5. Paste into your site's custom code section
